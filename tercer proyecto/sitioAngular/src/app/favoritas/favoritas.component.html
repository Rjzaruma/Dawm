Métodos del API Rest
Se utilizó GET para recuperar información de la base de datos. Posteriormente se definio diferentes parametros para filtrar la información. Entre ellos findAll() utilizado para traer todos los registros de las tablas y los que cumplieran ciertas condiciones como misma fase o rating. Por otro lado findOne() fue utilizado para traer la informacion específica de una pelicula y poder enlazarla con la tabla de actores.
